
# Cloud Enum
Init.
## Examples which worked
```bash
python3 cloud_enum.py -k target.com -k target
```
Using `-k` to give two different keywords. Can also give a *keyword file* with the `-kf` option.
### Quick Scan
```bash
./cloud_enum.py -kf keys -qs
```
Turns off mutations.
